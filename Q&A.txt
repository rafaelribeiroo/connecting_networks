<Redes de computadores>?
Basicamente, se dá por qualquer dispositivo que você tenha que acessa a
internet e que seja possível realizar cabeamento (ligar uma coisa na
outra e realizar troca de informações). Não necessariamente
precisamos de um dispositivo para ter uma rede, apenas
cabos (através de um crossover transferimos dados
entre pc's)

Tenho dois roteadores em casa e ainda assim está muito ruim o sinal, o que pode
ser?
Verifique se os dois estão em canais distintos, pode ser que estão na mesma
frequência, e um acaba atrapalhando o outro.

<LOOPBACK>?
Endereço da própria máquina

<DOWNSIZING>?
Antigamente, era apenas um processador centralizado para comandar todo o
tráfego de redes e informações, até o ponto que se tornou extremamente
lento, obrigando as redes a evoluir pro DOWNSIZING, que era o fato de
remover esse computador "grandão" e transformar em vários
computadores menores, exercendo funções diferentes
(servidores).

<GATEWAY>?
É o cara que representa o roteador que estamos conectado.

<ETHERNET>?
É a tecnologia da rede local (lan), basicamente serve pro modem falar com o
computador.

Difference between <MAN> e <WAN>
Se utilizarmos apenas uma tecnologia (não apenas ethernet, mas ADSL/...), é
uma MAN.

<Cabo blindado>?
A blindagem serve para combater as ondas eletromagnéticas, se trata de um meio
externo para não afetar como o sinal é transmitido.

Difference between <megabit> e <megabyte>
→ Mb: taxa de velocidade;
→ MB: armazenamento.

<Função WDS>?
Permite que um roteador converse com o outro a fim de criar uma rede única.

<PROXY>?
Você usa ele para agir entre seu "client" e sua conexão de internet, comumente
chamado de "man in the middle", ele vai servir para você caso queira
"filtrar" o acesso dos usuários, bloqueando alguns conteúdos. Uma outra
finalidade seria a solicitação de cadastramento para acessar a rede. Uma
alternativa de server proxy pra linux é o SQUID.

<DHCP>?
É o cara que distribui configuração de IP automaticamente.

<CLUSTER>?
É um nome dado a um grupo de computadores trabalhando em conjunto ou
arquitetura de sistema capaz de unir essas máquinas.
Temos diversos tipos:
    ← Cluster de alto desempenho: Focado em solucionar problemas que necessitam
    de muito processamento e entregá-los em um tempo considerável.
    ← Alta disponibilidade: Focado em manter a aplicação sempre no ar.
    ← Balanceamento de carga: Dividir as tarefas de forma similar entre nós.

<CRON>?
Serviço do sistema para trabalhar com tarefas agendadas.

Difference between <setup>, <bios> e <post>
Setup → Programa que vai embutido na placa para configurar seu computador;
BIOS → Identifica todos os componentes do computador (reconhecimento hardware);
Post → Testa todos os componentes do computador

<LEDs>?
É um diodo emissor de luz, a corrente que passa por ele é transformada em luz.

Difference between IP <público> e <privado>
O privado é pra rede interna, enquanto o público é para navegação.

<DISPOSITIVO>?
Um dispositivo é algo que podemos endereçar na rede.

<WORM>?
Vírus de computador que infecta todas as máquinas da mesma rede sem interação
do usuário. Se tiver tudo em uma rede só, infecta todos.
