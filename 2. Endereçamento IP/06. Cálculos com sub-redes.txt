Priori: Após aprendermos a calcular a quantidade de IPs endereçáveis e a
quantidade de sub-redes, vamos agora checar os endereços de rede, host e
broadcast.

Vamos trabalhar em cima do IP e máscara abaixo.

               192.168.10.1
              255.255.255.192

Faremos agora o cálculo de sub-rede para identificar quantas possibilidades
existem de endereço de rede, hosts endereçáveis e broadcast.

• Primeiro, identificamos o salto (quantidade de IPs que cada sub-rede terá).
    ← Para obter o salto, sempre definimos o limite de combinações subtraindo
pelo octeto distoante.
    ← 256 - 192 = 64
    ← Logo, o salto será 64.

•• Faremos agora a tabelinha para facilitar a visualização.

Visto todas as tabelas acima, aprenderemos agora um macete para realizar esses
cálculos de forma mais fácil.

                192.168.10.111
               255.255.255.192

===================================================================
• Converta a variação da máscara (192) pra binário:
    ← 11000000

•• Converta a variação do endereço IP (111):
    ← 01101111

••• Separando os bits uns dos zeros, ficará dessa forma:
    ← 11|000000
    ← 01|101111

    Para definir a sub-rede, mantenho os bits à esquerda e insere 0
    ← 01|000000: 64 → 192.168.10.64
    Caso fosse classe B, completamos com 0

    Para definir o broadcast, mesma coisa, completando com 1
    ← 01|111111: 127 → 192.168.10.127
    Caso fosse classe B, completamos com 255
